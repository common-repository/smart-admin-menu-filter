== Description ==
Quickly find the menu items you are looking for in the WordPress admin menu with this smart search filter.

= Features =
> WP Admin Menu Search
> Phonetic algorithm to capture misspellings and smarter filtering

== Installation ==
= Automatic installation =
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Category Posts Widget,

1. log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
2. In the search field type “Smart Admin Menu Filter” and click Search Plugins.
3. Once you’ve found plugin, you can install it by simply clicking “Install Now”.
4. Then, go to plugins page of WordPress admin activate the plugin.

= Manual installation =
1. Download the plugin.
2. Upload it to the plugins folder of your blog.
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
**Does this permanently change my admin menu?**
No the search modifies the menu temporarily while you are searching. Deleting your search term will bring the menu back to the original state.
